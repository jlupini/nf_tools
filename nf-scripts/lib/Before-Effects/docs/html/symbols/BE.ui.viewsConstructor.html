Class BE.ui.viewsConstructor

Takes a given window or palette object and assits in the creation of different views for the UI. The power comes in its ability to keep track of and give easy access to the hidden/shown views. Views are offered back to the user as a named array of references to the base Group object of each view.
Defined in: ui.jsxinc.
