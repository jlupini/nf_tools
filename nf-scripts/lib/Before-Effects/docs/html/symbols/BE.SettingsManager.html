Class Detail
BE.SettingsManager(sectionName, settings)
Provides a way to easily store and retrieve settings that After Effects should save. Settings are automatically saved on set and loaded on get. 
  var settingsManager = BE.SettingsManager("My Script Name",
      {
          "settingOne": "Setting One Value",
          "settingTwo": "Setting Two Value"
      }
  );
  settingsManager.get("settingOne");  //"Setting One Value"
  settingsManager.set("settingOne", "New Setting One Value");
  settingsManager.get("settingOne");  //"New Setting One Value");
Parameters:
{String} sectionName
The name that After Effects should use as this group of setting's sectionName. This can be the name of your script.
{Object} settings
An Object filled with key/value pairs to be saved within After Effect's settings. If this argument is not provided, settings will need to be added by using the add method.
Returns:
{SettingsManager} A BE.SettingsManager.
