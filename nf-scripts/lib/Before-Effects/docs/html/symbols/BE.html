<static> {boolean} BE.isDef(val)
Returns true if the specified value is not |undefined|.

WARNING: Do not use this to test if an object has a property. Use the in operator instead. Additionally, this function assumes that the global undefined variable has not been redefined. 

var x;
BE.isDef(x); //false
x = 1;
BE.isDef(x); //true

//To check if a object has a property use this instead:
var x.blah = 1;
"blah" in x; //true
"bam" in x; //false
Parameters:
{*} val
Variable to test.
Returns:
{boolean} Whether variable is defined.

<static> {boolean} BE.isNull(val)
Returns true if the specified value is truly |null|.

NOTE: There are a couple of things to look out for when working with null values:

  • typeof null returns 'object' instead of 'null'.
  • null == undefined returns true

BE.isNull(x); //true
!BE.isNull(x); //false

Imports the module defined by the given name path if it has not already been defined; If the module cannot be found, an exception is raised.

<static> {Boolean} BE.runScript(path)
Evaluates the script that resides at the given file path relative to the BE.BASE_PATH.

NOTE: This method does not check to see if the file has already been evaluated. 

BE.runScript("comp/comp.jsxinc"); //runs BE.BASE_PATH/comp/comp.jsx
}
Parameters:
{String} path
The file path for the script to be run.
Returns:
{Boolean} True if the file was succesfully evaluated, false if it wasn't.
