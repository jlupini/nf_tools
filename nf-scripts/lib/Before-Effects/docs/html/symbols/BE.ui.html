Aids in the creation of a label/input pair.

An object filled with properties that describe different aspects or values within this label/input pair.
{String} d.groupName
The name to give the group.
{String} d.labelName
The name to give the label of the input..
{String} d.inputName
The name to give the input field.
{String} d.groupOrientation Optional, Default: "row"
The orientation of the group. Possible values are 'row' or 'col'.
{String} d.groupMargins Optional, Default: 0
The margins to set for the group.
{String} d.groupAlignment Optional, Default: "['fill','fill']"
The alignment to use for the group.
{String} d.inputText Optional, Default: ""
The text to use as the input's value.
{String} d.inputMinSize Optional, Default: "[]"
The minimum size for the input field.
{String} d.inputJustify Optional, Default: "right"
How the label text is to be justified.
{String} t Optional, Default: "EditText"
The type of input to use. Ex. DropDown, EditText.
{String} extraRes Optional, Default: ""
A resource string that is inserted after the input field.
Returns:
